**Q：打开 `api/auth/session` 页面显示是空的怎么办？**
**A：** 说明您没有登录成功。请返回官网登录，然后再刷新该页面获取代码。

**Q：提示 Token 无效？**
**A：** 通常是因为复制不全。请回到前面，确保全选复制了所有内容。

**Q：提示充值失败？（极少数）**
**A：** 再次重新验证 Token。
